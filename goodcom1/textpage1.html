1. 입력

							
1) 방법 : 입력할 셀에 셀포인터를 놓고 - 데이터 입력 - [ENTER] 또는 [방향키]를 누른다. 
          [파일탭]-[옵션]-[고급]-ENTER를 누른후 다음셀로 이동 - 오른쪽 을 해보자.
2) 데이터 형식에 따른 입력방법							
	날짜 : 년,월,일 구분 기호를 - 또는 / 를 사용한다. 				
	시간 : 시간과 분 그리고 초 구분 기호를 :를 사용 --> 시:분:초			
	날짜와 시간을 동시에 : 날짜와 시간 사이를 한칸 띄우고 입력			
	분수 : 0 을 입력하고 한칸을 띄운 다음 분자/분모 를 입력한다.			
	숫자를 문자화 : 작은은 따옴표(')를 입력하고 숫자를 입력한다.			
	한자 : 한글을 입력하고 - [한자]키 - 해당 한자 선택				
	특수문자 입력 방법 : 한글 자음을 입력하고 [한자]키를 누른다음 해당 문자를 선택한다.
        (ㅁ : ※☆○◎◆▲♠☎☞㉿   ㅅ : ㉠㉮㈀㈎    ㅇ : ⓐ①⒜⑴)
3) 기타 유용한 입력 기능							
	하나의 셀에 여러 줄 입력 : ALT+ENTER						
	여러 셀에 같은 데이터를 동시에 입력 : 블록지정 -  데이터입력 - CTRL+ENTER	
	채우기 : 채우기핸들 - 드래그(채우기)						

2. 블록 설정 및 해제							
1) 용도 : 여러 셀을 동시에 서식을 편집하거나 내용을 삭제하는 작업을 일괄적으로 할 때 필요
2) 방법	
   연속적인 범위 블록		
        마우스로 드래그				
	SHIFT키를 누르고있으면서 방향키 클릭				
	시작 셀을 클릭한 후에 SHIFT키를 누른채 마지막 셀을 클릭 				
   비연속적인 범위 블록		
        범위를 설정한 후에 CTRL키를 누르면서 다른 범위를 블록 설정				
3) 해제 : 임의의 셀을 클릭							
							
3. 셀 내용 수정하기							
1) 방법 : 	
   수식입력줄에 마우스를 클릭하여 변경						
   바로가기키 F2를 사용						
   수정할 셀에서 마우스 포인터 모양이 넓은 십자가가 될 때 더블클릭												
4. 셀 내용 지우기							
1) 지울 곳을 선택하고 [DELETE]를 누른다.
   모두(서식,메모 포함)지우기 하려면 [홈] - [편집] - [지우기] - [모두지우기] 한다.						
							
참고) 마우스 포인터 모양							
①  : 마우스를 셀의 중앙에 놓으면 나타남. 이때 드래그 하면 블럭지정이 된다.		
②  : 마우스를 선택한곳의 테두리 부분에 놓으면 나타남. 이때 드래그하면 이동된다.	
③  : 마우스를 선택한곳의 오른쪽 아래부분 점에 놓으면 나타남. 이때 드래그 하면 채우기가 된다.
